# Bike-Sales-Analysis-Excel Dashboard
## Project Overview
This project analyzes bike sales data to identify customer trends, purchasing behaviour, and revenue using Microsft Excel.

## Tools Used
Microsoft Excel
Pivot Tables
Charts & Slicers
Data Cleaning

## Key Insights
Middle aged customers recorded the highest number of bike purchases
Higher income levels strongly correlate with purchase behaviour
Short distance commuters (0-1 miles) are the most frequent buyers
Purchase volume decreases as commute distance increases
